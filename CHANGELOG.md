# Clash Command-Line Interface ChangeLog

- v0.1 2020/08/27
  1. commands added:
      - `restart {?screen|systemctl}` : restart clash client
      - `scan`                        : scan clash config files
      - `use {config_filename}`       : use specific config file
      - `help`                        : show help info
      - `version`                     : show version info
